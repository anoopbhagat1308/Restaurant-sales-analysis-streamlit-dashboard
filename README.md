# 🍽️ Restaurant Sales Analysis Dashboard

An interactive **Restaurant Sales Analysis Dashboard** built using **Python, Streamlit, Pandas, and Plotly**. This project transforms raw restaurant sales data into meaningful business insights through interactive visualizations and key performance indicators (KPIs).

---

## 📖 Project Overview

The **Restaurant Sales Analysis Dashboard** enables users to upload a restaurant sales dataset and explore important business metrics through an intuitive and interactive web application.

The dashboard helps answer business questions such as:
- Which menu items generate the highest revenue?
- Which payment method is most popular?
- What are the daily and monthly sales trends?
- Which time of the day generates maximum sales?
- What is the average order value?

This project demonstrates practical skills in **Data Analysis, Exploratory Data Analysis (EDA), Data Visualization, and Streamlit application development**.

---

## 🚀 Features

- 📂 Upload Restaurant Sales CSV dataset

## 📂 Dataset Information

The dataset contains restaurant transaction details with the following columns:

| Column Name | Description |
|-------------|-------------|
| `order_id` | Unique Order ID |
| `date` | Date of Transaction |
| `item_name` | Name of Food Item |
| `item_type` | Food Category |
| `item_price` | Price of Item |
| `quantity` | Quantity Sold |
| `transaction_amount` | Total Bill Amount |
| `transaction_type` | Payment Method |
| `received_by` | Staff Member Receiving Order |
| `time_of_sale` | Breakfast / Lunch / Dinner |
